You can try the EPS using Amazons "pay by the hour" Virtual Server service called EC2. EC2 will provide you with a dedicated test server without the need to find the hardware resources internally.

When you have registered for the EC2 service you will need to start an EC2 virtual server using the AMI labelled ami-1c26c175 and a security group which allows connections on ports 80 (HTTP). Once the image has started you should be able to access the EPS using the URL http://[public DNS name of EC2 server]/passwordsafe/ (e.g. http://ec2-174-129-180-130.compute-1.amazonaws.com/passwordsafe/). This will present you with the login page where you can login using the user name admin and the password admin..

When you log in you will be asked to enter a license key, you can use the one month free trial key available at http://www.enterprise-password-safe.com/downloads/EPS/1.58/licence/licence.txt.

Once the license key has been entered the installation will be full operational.

If you need help using the EPS you can find the manual here and the support pages are here.

Please note: All of your data will be lost when you shut down the EC2 Virtual Server Instance. EC2 Virtual Servers can be left running for days, weeks, or even months, but the AMI is designed for demonstration purposes only, so please do not store any data in it which you do not have backed up or stored elsewhere.
